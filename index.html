Ordenagailu sare txiki baten oinarrizko egitura bat prestatu beharko dugu denda bateko bulego txiki baterako.

Baita web orri bat eta datu base basiko bat. Informatika enpresa bat gara eta gure enpresarentzat izen bat asmatu ondoren bulego txiki bat muntatzeko eskaera heldu zaigu. Demagun denda batek bulego txiki bat muntatu nahi duela baina jakintza gutxi daukatela informatikari buruz. Gure enpresara laguntza eske etorri dira eta beraien behar guztiak bete beharko ditugu.

Zergatik hau?

Bulego hori muntatzeko behar den informazioa jaso beharko dugu eta soluzio bat proposatu beharko diegu. Dendako arduradunak gure proposamena onartu ondoren soluzioa inplementatu behar dugu. Bulegoa montatzeaz gain dendarentzat web orri bat sortu beharko dugu eta datu base txiki bat egin beharko diogu dendako langile, produktu eta salmentak gordetzeko. Bestalde, prebentzio alorrari garrantzi berezia eman nahi diogunez, gai honetan ere betebehar batzuk eskatuko zaizkigu. Egindako guztia azalpena txosten batetan jarri beharko dugu eta aurkezpen bat egin beharko dugu erronka amaieran.
